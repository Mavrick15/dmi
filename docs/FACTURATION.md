# Facturer un patient (Consultation, Examen, Traitement)

## Où créer une facture

1. Aller dans **Tableau de bord** → **Facturation** (ou **Opérations financières**).
2. Onglet **Factures**.
3. Cliquer sur **Créer une facture** (ou **Nouvelle facture**).

## Étapes pour créer une facture

### 1. Choisir le patient
- Sélectionner le **patient** dans la liste.

### 2. Selon ce que vous facturez

- **Consultation**  
  - Choisir **Consultation liée (optionnel)** : sélectionner la consultation à facturer (date + motif).  
  - Saisir le **Montant total** (ex. tarif consultation).  
  - Optionnel : préciser dans **Type de facture** : « Consultation » et dans **Notes** le détail.

- **Examen(s) (analyses)**  
  - Dans **Analyses à facturer (optionnel)** : cocher une ou plusieurs **analyses terminées** non encore facturées.  
  - Saisir le **Montant total** (total des examens).  
  - Optionnel : **Type de facture** : « Examen(s) ».

- **Traitement**  
  - Ne pas lier de consultation ni d’analyses.  
  - Saisir le **Montant total** du traitement.  
  - Dans **Type de facture** choisir « Traitement » et dans **Notes** indiquer le type (ex. « Kinésithérapie », « Injection »).

### 3. Compléter la facture
- **Date d’émission** (par défaut : aujourd’hui).  
- **Date d’échéance** (optionnel, sinon 30 jours après l’émission).  
- **Montant déjà payé** (0 si aucun acompte).  
- **Notes** : précisions éventuelles.

### 4. Enregistrer
- **Aperçu** puis **Confirmer et créer**.

## Enregistrer un paiement

- Dans la liste des factures : cliquer **Payer** sur une facture en attente.  
- Saisir le **montant** et la **méthode de paiement** (espèces, carte, chèque, virement).  
- Le solde de la facture est mis à jour ; si le total est atteint, la facture passe en **Payée**.

## Récapitulatif

| Type        | Action dans « Créer une facture »                          |
|------------|-------------------------------------------------------------|
| Consultation | Lier une **consultation** + montant total                   |
| Examen(s)    | Cocher les **analyses** à facturer + montant total          |
| Traitement   | Aucune liaison ; **montant total** + notes si besoin       |

Les trois cas utilisent la même **Création de facture** ; seuls le lien (consultation/analyses) et le libellé (type + notes) changent.
